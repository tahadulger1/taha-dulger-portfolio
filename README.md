# Taha Dülger - Portfolyo

Modern React ve Next.js odaklı portfolyo web sitesi.


### Yerel Geliştirme

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Build al
npm run build

# Build'i önizle
npm run preview
```



## Teknolojiler

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons
