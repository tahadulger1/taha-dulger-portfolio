Taha Dülger - Portfolyo Sitesi Bu depo, React ve Vite (veya Next.js) kullanarak geliştirdiğim kişisel portfolyo sitemin kaynak kodlarını içermektedir.

Kurulum ve Yerel Çalıştırma Projeyi bilgisayarınızda çalıştırmak için aşağıdaki komutları sırasıyla uygulayabilirsiniz:

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Build al
npm run build

# Build'i önizle
npm run preview
Teknolojiler
React 18
Vite
Tailwind CSS
Lucide React Icons
