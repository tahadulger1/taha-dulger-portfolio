# Taha Dülger - Portfolyo

Modern React ve Next.js odaklı portfolyo web sitesi.

## 🚀 Hızlı Başlangıç

### Yerel Geliştirme

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Build al
npm run build

# Build'i önizle
npm run preview
```

## 📦 Deployment

### Vercel ile Deploy (Önerilen)

1. **GitHub'a Push Edin:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/tahadulger1/taha-dulger-portfolio.git
   git push -u origin main
   ```

2. **Vercel'e Giriş:**
   - https://vercel.com → GitHub ile giriş yapın
   - "Add New Project" → Repository'nizi seçin
   - Framework: **Vite** seçin
   - Deploy butonuna tıklayın

3. **Hazır!** 🎉
   - Her push'ta otomatik deploy
   - Ücretsiz SSL
   - Global CDN

Detaylı deployment rehberi için `DEPLOYMENT.md` dosyasına bakın.
